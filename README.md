# Superstore Data Analysis

## 📌 Project Overview
This repository contains a **Power BI dashboard** and the associated dataset used for analyzing sales performance of a fictional (or real) superstore. The goal of the project is to provide stakeholders with interactive visualizations that reveal insights into sales trends, product performance, customer behaviour, and other key business metrics.

## 📁 Repository Contents
- **Power BI Dashboard (.pbix):** A fully developed report featuring multiple pages with charts, slicers, maps, and KPIs.
- **Data File:** The underlying dataset (typically in `.xlsx`, `.csv`, or similar format) used to populate the dashboard.

> The exact filenames may vary; look for files containing `Superstore` in the name.

## ✨ Key Features
- Sales performance by region, category, and sub-category
- Profit analysis across product lines
- Time-based trends (yearly, quarterly, monthly)
- Customer segmentation and order patterns
- Interactive filters and drill-through capabilities for detailed exploration

## 🚀 Getting Started
1. **Prerequisites:**
   - Power BI Desktop installed (compatible version).
2. **Open the Dashboard:**
   - Launch Power BI Desktop and open the `.pbix` file from this repository.
3. **Refresh Data:**
   - If you've modified the data file or want to use the latest dataset, click **"Refresh"** in Power BI after verifying the data source path.

## 🛠 Customization & Extension

- Add new visuals or pages to the report to explore additional business questions.
- Use Power Query or DAX to create custom measures and calculated columns.
